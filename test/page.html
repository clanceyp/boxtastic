Before you start

CSS (cascading style sheets)

If you get asked to write any CSS the house style is alphabetical, use the wild card * to match any element and prefix free (don't bother prefixing CSS3 styles), e.g. 

*.mystyle {
    border: ... ;
    cursor: ... ;
    transform: ... ;
    z-index: ... ;
}

HTML (hypertext markup language)

Take care when answering questions related to HTML that you note which version of HTML your answer should be in.

The house style for HTML5 is to close tags and double quote element attributes e.g. 

<!DOCTYPE html>
    <html class="nojs"></html>
